## LetsFind – AI-Powered Phone Number Fraud Detection Platform ##

## Overview

Find is a data-driven fraud prevention application that allows users to check the trustworthiness of unknown phone numbers before sending digital payments.

The platform generates a **risk score (0–100)** using community reports and machine learning-based fraud prediction.

## Features

- Phone number search and verification
- Fraud risk score visualization
- Community fraud reports and comments
- Risk classification (Safe / Suspicious / High Risk)
- AI-based scam detection
